For API Documentation and How to run Project :- https://github.com/Vyomrana02/java-rest-api

For setuping DB:-
- Assuming CouchBase Server running.
- Create an Bucket on the server of any name. ( I have made buket of name :- employeewise)
- Go to java-rest-api\src\main\java\com\example\demo\com\example\demo\config\CouchbaseConfiguration.java and change backet name, username 
  and password of couchbase server according to your server username and password.

For using EMAIL Functionality
- Go to java-rest-api\src\main\resources\application.properties
- Write your own username and password of working Gmail Account.
